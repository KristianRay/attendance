# Attendance — Discord Community Server Website

A website for the **Attendance** Discord community server. Built for gamers and friends who want a fun and welcoming space to connect, play, and hang out.

🔗 **Live Demo:** [attendance-264w.vercel.app](https://attendance-264w.vercel.app)

---

## Pages

- **Home** — Hero video, activity grid
- **Info** — About the server and founding members
- **Activities** — Gaming, Voice Chats, and Fun sections with video previews
- **Rules** — Server guidelines
- **Feedback** — Google Form embed for community feedback
- **404** — Custom not found page

## Tech Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [React Router DOM](https://reactrouter.com/)
- [React Icons](https://react-icons.github.io/react-icons/)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

Deployed on **Vercel** via GitHub. Includes `vercel.json` for client-side routing support.
